📊 Beklenen Veritabanı Yapısı

Veritabanı Adı: toplanti_yonetim_sistemi

Host: localhost

Kullanıcı: root

Şifre: (boş veya belirlediğiniz)

📋 Gerekli Tablolar

1. katilimcilar Tablosu

  • id - INT, Primary Key, Auto Increment
  • ad_soyad - VARCHAR(255), NOT NULL
  • email - VARCHAR(255), NOT NULL, UNIQUE
  • telefon - VARCHAR(20)
  • kayit_tarihi - TIMESTAMP, DEFAULT CURRENT_TIMESTAMP

2. toplantilar Tablosu

  • id - INT, Primary Key, Auto Increment
  • toplanti_adi - VARCHAR(255), NOT NULL
  • tarih - DATE, NOT NULL
  • saat - TIME, NOT NULL
  • yer - VARCHAR(255), NOT NULL
  • durum - ENUM('planlandi', 'devam_ediyor', 'tamamlandi', 'iptal_edildi')
  • aciklama - TEXT
  • olusturma_tarihi - TIMESTAMP, DEFAULT CURRENT_TIMESTAMP

3. toplanti_katilimcilari Tablosu

  • id - INT, Primary Key, Auto Increment
  • toplanti_id - INT, Foreign Key (toplantilar.id)
  • katilimci_id - INT, Foreign Key (katilimcilar.id)
  • katilim_durumu - ENUM('bekliyor', 'onaylandi', 'reddedi')
  • kayit_tarihi - TIMESTAMP, DEFAULT CURRENT_TIMESTAMP

🚀 PHP Kurulum Adımları

  1. PHP İndir: https://windows.php.net/download/ adresinden "VS16 x64 Thread Safe" ZIP'ini indir
  2. Çıkart: ZIP'i C:\php klasörüne çıkart
  3. Konfigürasyon: php.ini-development dosyasını php.ini olarak kopyala
  4. PATH Ekle: Sistem PATH'ine C:\php ekle
  5. Test Et: Terminal'de "php -v" komutunu çalıştır
  6. Sunucu Başlat: Proje klasöründe "php -S localhost:8000" komutunu çalıştır

🔧 Alternatif Çözüm: XAMPP

PHP kurulumu karmaşık geliyorsa, XAMPP kullanabilirsiniz:

  1. https://www.apachefriends.org/ adresinden XAMPP indir
  2. Kurulum yap ve Apache + MySQL başlat
  3. Proje klasörünü C:\xampp\htdocs\ altına kopyala
  4. http://localhost/toplanti-yonetim-sistemi/ adresinden eriş
